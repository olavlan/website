# Website-specific instructions

## Development

* Run `gleam` and `lustre-dev-tools` to see a list of development commands you can run.
* Always follow official `lustre` examples, documentation and recommendations.
* Verify your work with type-checking, and testing in the browser that new features work as expected.
* To test and debug in the browser, use chrome-devtools and navigate to localhost:1234. Important: you cannot run `just serve` as a background task, so you have to ask the user to start the server if it's not running already.
